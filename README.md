🧠 **Breast Cancer Classification using Machine Learning**

This project focuses on building a predictive model to classify **breast cancer** tumors as **malignant or benign** using Machine Learning techniques.

📊 **Dataset**

It loads a **ready-made** dataset on breast cancer from the **sklearn** package into the **datasets** module.
